# 💻 Exercícios de Lógica de Programação

Projeto desenvolvido com o objetivo de praticar lógica de programação utilizando **JavaScript** e organizar a evolução dos estudos de forma estruturada.

🌐 **Acesse o site online:**  
👉 https://luanlhp777.github.io/exercicios/

---

## 🎯 Objetivo

- Praticar lógica de programação
- Desenvolver raciocínio algorítmico
- Aplicar estruturas como:
  - `for`
  - `while`
  - `do...while`
  - `if / else`
- Organizar exercícios de forma profissional
- Evoluir no uso de Git e GitHub

---

## 🧠 Conteúdos Praticados

- Estrutura de repetição (`for`, `while`, `do...while`)
- Estruturas condicionais
- Operações matemáticas
- Variáveis e entrada de dados
- Lógica de programação aplicada

---
